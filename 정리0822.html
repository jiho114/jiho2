form요소(input)

입력창 만들기 input태그를 활용한 입력 가능한 칸을 만들 수 있습니다.
비밀번호: 이메일:

form : 폼의 범위를 표시합니다.

fieldset : form 내 관련 컨트롤을 그룹으로 묶을 때 사용합니다.

legend : fieldset의 이름을 설정할 때 사용합니다.

input : 데이터 입력, 편집을 위한 컨트롤을 생성합니다. type과 함께 사용.

input태그의 type속성 활용

type=button
type=checkbox 체크박스입니다.
type=file 파일 업로드 버튼입니다.
type=submit 전송용으로만 사용하는 버튼입니다.
type=date 년/월/일 선택 버튼입니다.
type=number 숫자 생성 버튼입니다.
